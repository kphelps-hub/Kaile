Ashes: Reborn Data Pipeline

Ashes: Reborn is a competitive card game created by Plaid Hat Games. Online tournaments were regularly organized and run online. This project uses R to pull data automatically from googlesheets to track standings and statistics for the league. Data is loaded into an AWS server and shared with the community in googlesheets.

  • GoogleSheets
  • Git-Hub

Project Steps

  1. Deck and game information is compiled using R script from self-reported googlesheets.
  2. Data is summarized and saved into AWS.
  3. A copy of the data is put into a shared pivot table in googlesheets. The shared sheet shows commonly used cards and heroes in addition to player statistics.
