Consumer Expenditure Survey Analysis

In this project, I gathered data from the Bureau of Labor Statistics to analyze trends in consumer expenditure before and after the COVID-19 lockdowns. A full walk through and report can be found here. All code and data is stored in git-hub.

  • R Markdown Report
  • Git-Hub

Project Steps

  1. Collected data through use of the BLS API.
  2. Cleaned and analyzed data using R.
  3. Visualized the key takeaways using the ggplot package.

Results

Consumer spending has surpassed projections from before the lockdown.

Travel expenditure has increased across all age groups since the lockdown.

Consumers are changing their spending habits at a higher rate than previously.
